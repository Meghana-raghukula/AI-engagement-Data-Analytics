# AI-engagement-Data-Analytics
Developed an AI Video Engagement Analytics Dashboard using Python, SQL, and Tableau. Analyzed video engagement data to track key metrics such as total views, engagement rate, watch time, and completion rate. Created interactive visualizations to identify top-performing content and understand user engagement patterns.

# AI Video Engagement Analytics Dashboard

## 📌 Project Overview
This project analyzes video engagement data to understand user behavior and content performance. The goal is to track key metrics and generate insights using data visualization.

## 🛠️ Tools & Technologies
- Python (Pandas)
- SQL
- Tableau

## 📊 Key Features
- Tracks Total Views, Engagement Rate, Watch Time, and Completion Rate
- Interactive Tableau dashboard
- Visual analysis of content performance and user engagement

## 📈 Insights
- Identified top-performing video categories
- Analyzed engagement trends across different platforms
- Observed patterns in watch time and completion rates

- Video engagement dataset (CSV format)
